Parent-Student Links


POST
/api/v1/parent-student-links/generate-code
Step 1 — Generate link code (Student)


Generates an 8-character uppercase alphanumeric code valid for 24 hours. The student shares this code with their parent so the parent can connect. If an unexpired code already exists, it is returned without creating a new one.

Step 1.5 — Send code via SMS/Zalo (Student)

POST
/api/v1/parent-student-links/connect
Step 2 — Connect by code (Parent)


Validates the 8-character link code, verifies it has not expired, and establishes a verified parent-student relationship. The code is consumed on success and cannot be reused.
